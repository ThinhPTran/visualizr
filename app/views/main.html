Getting started

About the project

G3 is a Javascript charting library, built on top of d3.js. It's primary purpose is to allow easy access to dynamic in broswer geo-oriented charts.

G3 uses d3.js and jQuery as dependencies.

Quick Start

Finish this section when we are ready to push and have a bower and cdn version of G3 registered.

Plots

g3.plot

Example

Plots can have a width set explicitly or will size based off of their container div

var myPlot = g3.plot('plot_div')
  .height(400)
  .xDomain([0, 500])
  .yDomain([0, 300])
  .draw();

Logs

g3.log

Example

var myLogPlot = g3.plot('.log_plot_div')
  .height(400)
  .xDomain([0, 300])
  .yDomain([0, dataArray.length])
  .draw();

var myLog = g3.log(myLogPlot, dataArray).draw();

var myLog2 = g3.log(myLogPlot, dataArray)
   .color("black")
   .xMult(0.5)
   .xTrans(200)
   .draw();
